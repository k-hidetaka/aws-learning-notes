# EC2基礎メモ

## EC2とは
- Elastic Compute Cloudの略
- クラウド上で仮想サーバー(インスタンス)を必要な分だけ借りられるサービス

## 主要な概念
- **インスタンスタイプ**: CPU・メモリなどのスペックの組み合わせ(例: t2.micro, m5.large)
- **AMI (Amazon Machine Image)**: インスタンスを起動する際の元になるテンプレート(OSやミドルウェアの構成情報)
- **EBS (Elastic Block Store)**: EC2にアタッチするストレージ(ボリューム)

## 料金モデル
| モデル | 特徴 |
|---|---|
| オンデマンド | 使った分だけ課金、柔軟性が高い |
| リザーブド | 長期利用前提で割引 |
| スポット | 余剰キャパシティを安く使える、中断のリスクあり |
