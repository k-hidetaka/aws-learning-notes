# Docker基礎メモ

## Dockerとは
- コンテナ型仮想化技術
- OSレベルで環境を隔離し、アプリとその依存関係をまとめてパッケージ化する

## よく使うコマンド
| コマンド | 説明 |
|---|---|
| `docker build -t <name> .` | Dockerfileからイメージを作成 |
| `docker run <image>` | イメージからコンテナを起動 |
| `docker ps` | 起動中のコンテナ一覧 |
| `docker images` | ローカルのイメージ一覧 |
